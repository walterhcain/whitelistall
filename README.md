# Whitelist All
Whitelisting plugin to mass whitelist steam64 IDs for Unturned servers.

## How it works
1. Upon starting the server with the plugin, a whitelist file will be made in the directory. 
2. Steam64 IDs can be placed in the whitelist document, each on new lines.
3. From console or within the server, you can use the whitelist all command and the game will mass whitelist any of the IDs in the server.

## Commands
  ```/whitelistall``` - Whitelist all Steam64 IDs in the whitelist document<br>
  ```/wa``` - Alias of the above command for quicker typing

## Permisions
  Any admins can use the command. Otherwise, the following permission is needed to be added to the group or player.<br>
    ```wc.whitelistall```
